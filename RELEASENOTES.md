# Release Notes - Version 2.2 (Code 13)

## What's New

- **Midnight & Teal Theme**: Replaced the purple/violet theme with a premium Midnight Blue and Teal palette for a more professional feel.
- **Interactive Presets**: You can now toggle specific sound categories directly on the Preset cards before starting a session.
- **Redesigned Icons**: Preset cards now feature color-coded category icons for better scannability.
- **Improved Tab Switching**: Smooth transitions between Manual and Preset modes.
